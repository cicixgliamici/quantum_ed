# Entanglement

Two-qubit states live in a 4D space: \(\mathbb{C}^2 \otimes \mathbb{C}^2\).

A classic example is the Bell state:
\[
|\Phi^+\rangle = \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)
\]
It cannot be written as \(|\psi\rangle\otimes|\phi\rangle\) (not separable).

We will later use:
- reduced density matrices (partial trace)
- entanglement entropy (intro level)
