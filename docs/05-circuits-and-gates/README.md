# Circuits & Gates

Gates are unitary matrices applied to state vectors.

Common 1-qubit gates:
- X (NOT), Y, Z (Pauli)
- H (Hadamard)
- S, T (phase gates)

2-qubit gates:
- CNOT
- CZ

A circuit is a composition (matrix product) of gates.
